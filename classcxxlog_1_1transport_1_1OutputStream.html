Detailed Description

A class for logging messages to an output stream with timestamp and severity level.

This class provides a way to log messages to an output stream with a timestamp and a specified severity level. The log messages are formatted in the following way: "<timestamp> <severity>: <message>\n"

This function logs a message with a specified severity level and includes a timestamp indicating when the message was logged. The message is written to the output stream provided in the constructor.

Note
The timestamp is UTC in the format "YYYY-MM-DDTHH:MM:SS".
